Frequently Asked Questions

Find answers to the most common questions about LyBotic

How do I add LyBotic to my server?

To add LyBotic to your Discord server:

  1. Click the "Add to Server" button on this website
  2. Select your server from the dropdown
  3. Review and approve the requested permissions
  4. Click "Authorize" to complete the setup

Once added, type !help to get started!

What permissions does LyBotic need?

LyBotic requires the following permissions for full functionality:

  • Read Messages - To see commands
  • Send Messages - To respond to commands
  • Manage Messages - For moderation features
  • Embed Links - For rich message embeds
  • Attach Files - For image and file features
  • Use External Emojis - For reaction commands
  • Connect & Speak - For music features
  • Manage Roles - For role management commands

You can customize permissions per channel if needed.

How do I use commands?

LyBotic supports both prefix and slash commands:

Prefix Commands:

Type ! followed by the command name. For example:

  • !help - Shows the help menu
  • !ban @user - Bans a user
  • !play song name - Plays music

Slash Commands:

Type / and select from the autocomplete menu.

Visit our Commands page for a complete list.

Can I change the command prefix?

Yes! You can customize the command prefix using the web dashboard:

  1. Go to the Dashboard
  2. Login with Discord
  3. Select your server
  4. Navigate to Settings → General
  5. Change the "Command Prefix" field
  6. Save your changes

The prefix can be 1-3 characters long.

Who can use moderation commands?

Moderation commands require appropriate Discord permissions:

  • Ban/Kick - Requires "Ban Members" or "Kick Members" permission
  • Mute/Timeout - Requires "Moderate Members" permission
  • Role Management - Requires "Manage Roles" permission
  • AutoMod - Requires "Manage Server" permission

Server administrators can always use all commands.

How do I set up AutoMod?

LyBotic integrates with Discord's native AutoMod system:

  1. Use !automod setup for quick configuration
  2. Or use the web dashboard for advanced settings
  3. Choose from preset filters (spam, bad words, etc.)
  4. Customize actions and log channels

AutoMod rules help automatically moderate your server 24/7.

The bot isn't responding to commands

If LyBotic isn't responding, try these steps:

  1. Check if the bot is online (green status)
  2. Verify the bot has "Read Messages" permission in the channel
  3. Make sure you're using the correct prefix (default is !)
  4. Check if the channel has any bot restrictions
  5. Try using the command in a different channel

If issues persist, join our Discord server for help.

Music isn't working

For music troubleshooting:

  1. Make sure you're in a voice channel
  2. Check if the bot has "Connect" and "Speak" permissions
  3. Verify the voice channel isn't user-limited
  4. Try a different song or platform
  5. Use !music status to check the player

Supported platforms: YouTube, Spotify, SoundCloud.

Economy features aren't working

Economy system troubleshooting:

  1. Use !balance to check your wallet
  2. Try !work to earn initial coins
  3. Check if economy is enabled: !economy status
  4. Review cooldown timers for commands

Your economy data is persistent across server restarts.
